# story_lingo-privacy

Story Lingo 的 **隐私政策** 与 **使用条款** 静态页，通过 GitHub Pages 发布。

**仓库：** https://github.com/happylife-dev/story_lingo-privacy

## 线上地址（启用 Pages 后）

在仓库 **Settings → Pages** 中选择 **main** 分支、`/ (root)` 作为源。

| 页面 | URL |
|------|-----|
| 首页 | https://happylife-dev.github.io/story_lingo-privacy/ |
| 隐私政策 | https://happylife-dev.github.io/story_lingo-privacy/privacy/ |
| 使用条款 | https://happylife-dev.github.io/story_lingo-privacy/terms/ |

## 从本目录推送

```bash
cd happylife-dev.github.io
git init
git add .
git commit -m "docs: Story Lingo 隐私政策与使用条款"
git branch -M main
git remote add origin https://github.com/happylife-dev/story_lingo-privacy.git
git push -u origin main
```

若远程已有提交且需覆盖，在确认安全后使用 `git push -u origin main --force`。
